title: View Instantiated Contracts
icon: list
---

To view contracts you've instantiated:

1. **Access the List**: Click on 'Instantiated by me' in the left sidebar
2. **Contract Overview**: Each contract displays its name, code ID, and whether it's tagged as a project.
3. **Actions**: Options to execute, query, or view each contract are available.
4. **Filtering**: Use search functionality to filter contracts based on criteria like contract name or code ID.

For full details and interactions to organize your contracts, See [Organize your contracts](https://docs.alleslabs.com/user/cosmwasm/contract/organize) with Celatone
