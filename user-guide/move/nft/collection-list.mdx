The **“NFT Collection List”** on Celatone serves as a single place and comprehensive list for users to explore various NFT collections available on chain with key detail like collection name and description

<Frame>
    <img src="/images/user-docs/move/nft/nft-collection-list.jpg" />
</Frame>


## Available Actions

- **Explore**: Users can briefly see collection cover image, name and description.
- **Search**: Users can search for collection using the collection name.
- **Clicking** the collection card navigates you to view the selected [Collection Details](https://docs.alleslabs.com/user-guide/move/nft/collection-list) page.
