# Setting up Dependencies

## Dependencies

- **Go**: Celatone's indexer uses Go 1.18. To install Go, follow the instructions at [golang.org/doc/install](https://golang.org/doc/install)
- **Node**: Install from either the official Node JS [website](https://nodejs.org/en) or using [nvm](https://github.com/nvm-sh/nvm)
- **Yarn**: Follow instructions from the official Yarn [website](https://classic.yarnpkg.com/lang/en/docs/install)
- **Docker**: Follow instructions from the official Docker [website](https://docs.docker.com/get-docker/)

## Celatone Components

## Indexer

Clone the Celatone Osmosis indexer repository from [alleslabs/osmosis](https://github.com/alleslabs/osmosis).

```bash
git clone https://github.com/alleslabs/osmosis
```

## Frontend

Clone the Celatone frontend repository at [alleslabs/celatone-frontend](https://github.com/alleslabs/celatone-frontend)

```bash
git clone https://github.com/alleslabs/celatone-frontend
```
